# REQUIREMENTS


* Ruby version

  ```
  2.7.1
  ```

* System dependencies

  ```
  Postgresql 12
  ```

* Configuration

* Database setup

  ```sh
  rails db:create
  rails db:migrate
  ```

* How to run the test suite

  ```sh
  rails test
  ```

* Deployment instructions

  ```
  heroku git:remote -a fec-online-judge
  git push origin heroku
  ```
