# REQUIREMENTS


* Ruby version

  ```
  2.6.3
  ```

* System dependencies

  ```
  Postgresql 11.5
  ```

* Configuration

* Database setup

  ```sh
  rails db:create
  rails db:migrate
  ```

* How to run the test suite

  ```sh
  rails test
  ```

* Deployment instructions

  ```
  heroku git:remote -a fec-online-judge
  git push origin heroku
  ```
